1989年3月12日、イギリスの計算機科学者「ティム・バーナーズ＝リー」がWWW（ワールドワイドウェブ）を構築するアイデアをまとめた 提案書 を発表したことがHTMLの起源とされています。

HTML は、ワールドワイドウェブ（通称ウェブ）を構築するために使う言語 ハイパーテキスト・マークアップ・ランゲージ の略語。

自分が作った文書に関係する文書やファイルへのハイパーリンク （自分が保管している文書・ファイルだけじゃなく、インターネット上にある他人のファイルともリンクできるくらいハイパーなリンク） を含むことができるほどのハイパーな文書 （ハイパーテキスト） を、視覚・聴覚・触覚情報として「タグ」を用いてマークアップ する言語という意味で HTML と名付けられたそうです。

HTMLファイル は、HTMLを表示するための専用ブラウザ （インターネットエクスプローラー、サファリ、クロームなど） で閲覧でき、常時インターネットに接続されたコンピューター（サーバー）に公開された HTMLファイル はハイパーリンクで繋がり、世界中のだれもがアクセスできる情報網「ワールドワイドウェブ」が構築されます。

その後1993年に、世界で初めてインターネット上にウェブサイト info.cern.ch が公開されてからこれまでの間、HTMLは幾度となくバージョンアップを繰り返し、今ではアプリケーションの開発などにも使える言語にまで進化し続けています。
そして2021年1月29日、Web Hypertext Application Technology Working Group（WHATWG、ワットワーキンググループ、ワットダブルジー） が新たに定義した HTML Living Standard がHTMLの新たな世界基準に認定されました。

※ ワットワーキンググループとは、Apple・Mozilla・Opera（後にMicrosoftも参加）の開発者たちによって結成されたHTMLの標準規格を作成する団体です。
